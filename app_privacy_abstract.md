# UCloud App隐私声明摘要
<br>UCloud（以下简称我们）会严格遵从国家的法律法规要求，将合法合规融入产品设计和业务流程中，并采用业界先进的安全技术保证数据安全。同时我们非常重视您的个人信息和隐私保护，致力于捍卫每个人的隐私权利，我们会遵循合法、正当、必要和诚信原则，具体包括：

<br>**目的明确、合理**：坚持处理个人信息应当具有明确、合理的目的，并应当与处理目的直接相关，采取对个人权益影响最小的方式
<br>**数据最小化**：坚持限于实现处理目的的最小范围收集个人信息，不过度收集个人信息
<br>**公开、透明**：坚持处理个人信息应当遵循公开、透明原则，公开个人信息处理规则，明示处理的目的、方式和范围
<br>**质量保证**：坚持处理个人信息应当保证个人信息的质量，避免因个人信息不准确、不完整对个人权益造成不利影响
<br>**安全保障**：坚持对我们个人信息处理活动负责，并采取必要措施保障所处理的个人信息的安全

<br>**UCloud App《隐私声明摘要》** 如下：
### 一、收集的个人信息类型  
| 信息名称         | 使用目的                     | 使用场景                                                                 |
|------------------|------------------------------|--------------------------------------------------------------------------|
| 姓名             | 实名认证                     | 用户进行实名认证时                                                       |
| 手机号码         | 创建账号、登录账号、账号安全 | 用户注册/绑定账号时；用户登录账号时                                      |
| 账号名           | 创建账号、登录账号、账号安全 | 用户注册、登录账号时；基本资料展示时                                     |
| 登录密码         | 登录账号、账号安全           | 登录账号时；用户修改登录密码时                                           |
| 电子邮件地址     | 账号安全                     | 用户设置/修改绑定邮箱时                                                  |
| <u>**身份证号**</u>         | 实名认证                     | 用户进行实名认证时                                                       |
| <u>**面部特征**</u>        | 实名认证、人脸校验、账号安全 | 用户进行实名认证时                                                       |
| 手机号           | 用户登录、账号注册和管理     | 用户每次填写并提交手机号时                                               |
| 搜索记录         | 搜索                         | 用户每次搜索时                                                           |

### 二、申请的权限  
| 权限信息 | 使用场景、目的                                                                 |
|----------|------------------------------------------------------------------------------|
| 相机 | 扫描Web端登录二维码时、<u>**扫描身份证**</u>进行实名认证时、绑定MFA时；<br>如您拒绝授权部分功能或服务所需信息，您将无法使用相关功能或服务，但这不影响您正常使用我们提供的其他功能或服务。 |
| 存储 | 从本地存储中上传<u>**身份证照片**</u>进行实名认证时，从从本地存储中上传二维码进行扫码登录或绑定MFA时；<br>如您拒绝授权部分功能或服务所需信息，您将无法使用相关功能或服务，但这不影响您正常使用我们提供的其他功能或服务。 |

### 三、您如何管理自己的信息
您在使用我们服务的过程中，可以通过查询、复制、更正、补充、删除、撤回授权、注销账号等方式来管理您的个人信息。
