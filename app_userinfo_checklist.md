# 个人信息收集清单
<br>感谢您选择和使用UCloud App！
<br>UCloud App将尽全力保护您的个人安全，并保障您在个人信息处理过程中的相关权益。为了向您提供UCloud App的基本功能及附加功能，我们需要收集您在使用服务时通过您主动授权/提供等方式产生的个人信息。以下为可能需要收集的个人信息清单：



| 信息名称         | 使用目的                     | 使用场景                                                                 |
|------------------|------------------------------|--------------------------------------------------------------------------|
| 姓名             | 实名认证                     | 用户进行实名认证时                                                       |
| 手机号码         | 创建账号、登录账号、账号安全 | 用户注册/绑定账号时；用户登录账号时                                      |
| 账号名           | 创建账号、登录账号、账号安全 | 用户注册、登录账号时；基本资料展示时                                     |
| 登录密码         | 登录账号、账号安全           | 登录账号时；用户修改登录密码时                                           |
| 电子邮件地址     | 账号安全                     | 用户设置/修改绑定邮箱时                                                  |
| 身份证号         | 实名认证                     | 用户进行实名认证时                                                       |
| 面部特征         | 实名认证、人脸校验、账号安全 | 用户进行实名认证时                                                       |
| 手机号           | 用户登录、账号注册和管理     | 用户每次填写并提交手机号时                                               |
| 上传图片信息     | 图片上传                     | 上传身份证照片进行实名认证时                                             |
| 搜索记录         | 搜索                         | 用户每次搜索时                                                           |
| 摄像头           | 扫描内容                     | 扫描Web端登录二维码时；扫描身份证进行实名认证时                          |
